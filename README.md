Оплата доставки при получении для Shop-Script
===============================================

[English version](README.en.md)

Плагин позволяет скрыть стоимость доставки из суммы к оплате онлайн: покупатель оплачивает только товары,
а доставку оплачивает курьеру при получении заказа.

При создании или редактировании заказа плагин обнуляет стоимость доставки и сохраняет исходную сумму во
внутреннем параметре заказа. Итоговая сумма заказа пересчитывается соответственно. Обнуление применяется
только для выбранных в настройках сочетаний способов доставки и оплаты. В бэкенде на странице заказа
отображается сохранённая стоимость доставки.

Уведомления о создании заказа отправляются до срабатывания плагина, поэтому в письме покупатель увидит
исходную сумму с доставкой. На отображение стоимости доставки в процессе оформления заказа плагин не влияет —
он срабатывает только после его создания.

## Требования

- PHP 7.4 или новее
- Webasyst Framework (installer) 3.0 или новее
- Shop-Script 8.0 или новее (8.17 для использования хелпера)

Для шаблонов Smarty доступен хелпер, возвращающий сохранённую стоимость доставки:
`{$wa->shop->dbcPlugin->shippingCost($order)}`. Метод принимает ID заказа, массив данных заказа или объект
`shopOrder`. Требуется Shop-Script 8.17 или новее.
